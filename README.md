# Eye-Tracking Advertising Perception Experiment

Διπλωματική εργασία: Μελέτη της αντίληψης των χρηστών για διαφορετικούς τύπους διαδικτυακών διαφημίσεων με χρήση eye-tracking.

## Περιγραφή

Αυτό το πείραμα χρησιμοποιεί WebGazer.js για να καταγράψει τα δεδομένα βλέμματος των συμμετεχόντων ενώ αλληλεπιδρούν με 9 διαφορετικά σενάρια διαφημίσεων pop-up.

## Τεχνολογίες

- **WebGazer.js**: Eye-tracking μέσω webcam
- **Vanilla JavaScript**: Λογική πειράματος
- **Google Apps Script**: Αποθήκευση δεδομένων

## Πώς να συμμετάσχω

1. Επισκεφθείτε το πείραμα: [Link]
2. Επιτρέψτε την πρόσβαση στην webcam
3. Ολοκληρώστε την βαθμονόμηση (9 σημεία × 5 κλικ)
4. Δείτε τις 9 διαφημίσεις και απαντήστε στις ερωτήσεις
5. Υποβάλετε τις απαντήσεις σας

## Σημειώσεις

- Απαιτείται σύγχρονος browser (Chrome/Firefox/Edge)
- Χρειάζεται webcam και καλός φωτισμός
- Διάρκεια: ~10-15 λεπτά
- Τα δεδομένα είναι ανώνυμα
